02

基本功能

代理後台首頁佈局概覽與個人化權限設置說明

🖼️
一、界面佈局核心

下圖展示了代理看板的核心佈局，分為導航控制、帳號資訊與權限監控、以及數據看板三大區域：

區塊名稱 核心功能與說明
功能主選單 (左側) 導航核心：包含首頁、代理組織管理、統計報表、投注記錄、資金細目與轉帳等入口。
帳號與參數區 (頂部) 管理中心：顯示當前登錄者資訊、錢包餘額，以及核心運營權限（含當前額度、佔反比、有效投注比）。
數據儀表板 (中心) 營運總覽：呈現代理體系整體績效，包含總投注、輸贏、佣金損益等。
⚙️
二、個人化設置中心

1. 個人帳號資訊 (側邊欄/頂部)

顯示當前登錄的身份核心資訊：

  • 帳號名稱與等級：顯示如「leozdf1 [總代理]」。
  • 帳戶餘額：即時顯示目前可用金額，支持隱藏（點擊小眼睛）與手動刷新。
  • 設置入口：點擊齒輪圖標 ⚙️ 進入詳細功能清單。

2. 設置選單功能

包含以下核心操作選項：

  1. 語言切換：支持 簡中、英文、越南文、菲律賓文、韓文，點擊後即時套用。
  2. 夜間模式：一鍵切換深色/淺色皮膚，降低長時間操作的視覺疲勞。
  3. 修改密碼：定期更新登入密碼，維護帳號安全。
  4. 清除緩存：若遇到數據顯示異常，可清除本地暫存檔案。
  5. 登出：安全結束當前會話。
02

Basic Features

Overview of the Agent Dashboard layout and personalized settings.

🖼️
I. Layout Overview

The image below showcases the core layout of the dashboard, divided into three main zones: Navigation, Account Info & Permissions, and Data Dashboard.

Block Name Core Function & Description
Main Menu (Left) Navigation Hub: Entrance to Agent Management, reports, betting records, and funds management.
Account Info (Top) Personal Center: Displays login info (ID, level), balance, and operational parameters (Credit, Ratios).
Data Dashboard (Center) Performance Overview: Summarizes performance metrics such as turnover, win/loss, and commission data.
⚙️
II. Account Settings Center

1. Personal Account Info

Displays the core information of the currently logged-in identity:

  • Account & Level: Displays your ID and agent level (e.g., General Agent).
  • Balance: Real-time display of available balance. Supports hiding (eye icon) and manual refreshing.
  • Settings Entry: Click the gear icon ⚙️ to access the features list.

2. Settings Menu Items

Includes the following management options:

  1. Language Selection: Supports Multiple languages. Changes applied instantly.
  2. Night Mode: Toggle between dark and light themes for visual comfort.
  3. Change Password: Update your login credentials regularly for security.
  4. Clear Cache: Reset local data if you encounter display issues.
  5. Logout: Securely terminate your current session.
02

Chức năng cơ bản

Tổng quan về bố cục và cài đặt tài khoản trong bảng điều khiển đại lý.

🖼️
I. Tổng quan về bố cục

Hình ảnh dưới đây hiển thị bố cục cốt lõi của hệ thống đại lý, được chia thành ba khu vực chính: Điều hướng, Thông tin tài khoản & Quyền hạn, và Bảng dữ liệu.

Tên khối Chức năng & Mô tả chính
Menu chính (Bên trái) Trung tâm điều hướng: Truy cập Quản lý đại lý, báo cáo, lịch sử cược và quản lý quỹ.
Thông tin tài khoản (Phía trên) Quản lý tài khoản: Hiển thị ID người dùng, số dư và các quyền vận hành (Hạn mức, Tỷ lệ).
Bảng điều khiển dữ liệu (Trung tâm) Tổng quan hiệu suất: Tóm tắt hiệu quả hoạt động như doanh thu, thắng/thua và dữ liệu hoa hồng.
⚙️
II. Trung tâm cài đặt cá nhân

1. Thông tin tài khoản

Hiển thị thông tin cốt lõi của tài khoản đang đăng nhập:

  • Tài khoản & Cấp độ: Hiển thị ID và cấp đại lý (ví dụ: Tổng đại lý).
  • Số dư: Hiển thị số dư hiện tại theo thời gian thực. Hỗ trợ ẩn số tiền và làm mới thủ công.
  • Lối vào cài đặt: Nhấp vào biểu tượng bánh răng ⚙️ để xem danh sách chức năng.

2. Danh mục cài đặt

Bao gồm các tùy chọn vận hành sau:

  1. Chuyển đổi ngôn ngữ: Hỗ trợ nhiều ngôn ngữ, áp dụng ngay lập tức sau khi chọn.
  2. Chế độ ban đêm: Chuyển đổi giao diện Sáng/Tối để bảo vệ mắt.
  3. Đổi mật khẩu: Cập nhật mật khẩu thường xuyên để bảo mật.
  4. Xóa bộ nhớ đệm: Xóa tệp tạm thời nếu gặp lỗi hiển thị dữ liệu.
  5. Đăng xuất: Kết thúc phiên làm việc an toàn.

Mga Pangunahing Feature

Overview ng layout ng Dashboard at mga setting ng account sa backend.

🖼️
I. Pangkalahatang-ideya ng Layout

Ipinapakita ng imahe sa ibaba ang pangunahing layout ng dashboard, na nahahati sa tatlong zone: Navigation, Account Info & Permissions, at Dashboard ng Data.

Pangalan ng Block Pangunahing Gamit at Paglalarawan
Pangunahing Menu (Kaliwa) Navigation Hub: Pasukan sa Pamamahala ng Agent, mga ulat, kasaysayan ng taya, at pondo.
Info ng Account (Itaas) Personal Center: Ipinapakita ang ID, balanse, at mga parameter gaya ng Credit at Ratio.
Dashboard ng Data (Gitna) Performance Overview: Buod ng mga sukat gaya ng turnover, panalo/talo, at komisyon.
⚙️
II. Personal Setting Center

1. Info ng Personal na Account

Ipinapakita ang pangunahing impormasyon ng kasalukuyang account:

  • Account at Antas: Ipinapakita ang ID at level (hal. General Agent).
  • Balanse: Real-time na display ng balanse. Maaaring itago o i-refresh nang manual.
  • Settings Entry: I-click ang gear icon ⚙️ para sa listahan ng mga feature.

2. Mga Item sa Menu ng Settings

Kasama ang mga sumusunod na opsyon:

  1. Pagpili ng Wika: Sinusuportahan ang maraming wika. Agad na nailalapat.
  2. Night Mode: Magpalipat-lipat sa dark/light theme para sa visual comfort.
  3. Palitan ang Password: Mag-update ng password para sa seguridad.
  4. I-clear ang Cache: I-reset ang local data kung may problema sa display.
  5. Logout: Ligtas na tapusin ang iyong session.
