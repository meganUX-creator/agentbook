如何查看收益

了解如何在代理后台查看您的占成收益与下级投注数据

📊
方法一：首页仪表板概览

您可以在首页仪表板快速查阅当下的经营数据，图示说明如下：

  1. 侧边栏「首页」(标示 1)：点击即可进入概览页面。
  2. 查看详细信息 (标示 2)：点击可展开个人帐号详细资讯，包含：占成比、有效投注比、开户时间等。
  3. 选择查询时间 (标示 3)：利用日期选择器（今日、昨日、本周等）快速筛选数据统计范围。
  4. 数据仪表板 (标示 4)：概览当下各层级的代理/会员总数、投注总额、总输赢、应得佣金等关键数据。
💡
收益数据说明
字段 说明
有效投注额 下级所有有效投注的总金额（扣除对冲等无效注）
输赢金额 下级在该周期内的总输赢金额
占成收益 按您的占成比例计算所得的收益金额
结算状态 显示该周期是否已完成结算
✅
收益每日自动结算一次，结算时间通常为凌晨 00:00–02:00。结算完成后数据不可更改。

View Earnings

Learn how to view your commission earnings and downline betting data in the Agent Backend.

📊
Method 1: Home Dashboard Overview

You can quickly view the current operational data on the home dashboard. The illustration is explained as follows:

  1. Sidebar "Home" (Marked 1): Click to enter the dashboard.
  2. View Detailed Information (Marked 2): Click to expand account details, such as: Occupancy Ratio, Valid Bet Ratio, Account Creation Time, etc.
  3. Select Query Time (Marked 3): Use the date picker (Today, Yesterday, This Week, etc.) to quickly filter the range.
  4. Data Dashboard (Marked 4): An overview of key data for agents/members, total turnover, total win/loss, commissions, etc.
💡
Earnings Data Explanation
Field Description
Valid Bet Amount Total valid bets from downlines (excluding hedged or invalid bets)
Win/Loss Amount Total win/loss amount for downlines during this period
Commission Earnings Earnings calculated based on your commission ratio
Settlement Status Shows whether settlement has been completed for this period
✅
Earnings are automatically settled once daily, usually between 00:00–02:00. Data cannot be changed after settlement.

Xem thu nhập

Tìm hiểu cách xem thu nhập hoa hồng và dữ liệu cược của cấp dưới trong hệ thống đại lý.

📊
Cách 1: Tổng quan trang chủ

Bạn có thể nhanh chóng xem dữ liệu vận hành hiện tại trên bảng điều khiển trang chủ. Giải thích hình ảnh như sau:

  1. Thanh bên "Trang chủ" (Đánh dấu 1): Nhấp để vào bảng điều khiển.
  2. Xem thông tin chi tiết (Đánh dấu 2): Nhấp để mở rộng chi tiết tài khoản, như: Tỷ lệ chiếm thành, Tỷ lệ cược hợp lệ, Thời gian tạo tài khoản, v.v.
  3. Chọn thời gian (Đánh dấu 3): Sử dụng trình chọn ngày (Hôm nay, Hôm qua, Tuần này, v.v.) để lọc nhanh phạm vi dữ liệu.
  4. Bảng điều khiển dữ liệu (Đánh dấu 4): Tổng quan về các dữ liệu chính cho đại lý/thành viên, tổng doanh thu, tổng thắng/thua, hoa hồng, v.v.
✅
Thu nhập được quyết toán tự động mỗi ngày, thường vào 00:00–02:00. Dữ liệu không thể thay đổi sau khi quyết toán.

Tingnan ang Kita

Alamin kung paano tingnan ang iyong commission earnings at downline betting data sa Agent Backend.

📊
Method 1: Home Dashboard Overview

Mabilis mong makikita ang kasalukuyang operational data sa Home Dashboard. Ang paliwanag sa larawan ay sumusunod:

  1. Sidebar "Home" (Marka 1): I-click para pumasok sa dashboard.
  2. View Detailed Information (Marka 2): I-click para i-expand ang mga detalye ng account tulad ng: Occupancy Ratio, Valid Bet Ratio, Oras ng Paglikha ng Account, atbp.
  3. Pumili ng Oras ng Query (Marka 3): Gamitin ang date picker (Ngayon, Kahapon, Ngayong Linggo, atbp.) para mabilis na i-filter ang range.
  4. Data Dashboard (Marka 4): Pangkalahatang ideya ng mga pangunahing data para sa mga agent/miyembro, kabuuang turnover, kabuuang panalo/talo, mga komisyon, atbp.
✅
Ang mga kita ay awtomatikong nai-settle isang beses bawat araw, karaniwang sa pagitan ng 00:00–02:00. Ang data ay hindi mababago pagkatapos ng settlement.
